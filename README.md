# Portfolio Arielle Noutais

Portfolio ultra moderne et élégant pour Arielle Noutais - Développeuse Flutter Mobile.

## ✨ Caractéristiques

- **Design premium** : Minimaliste, luxueux, glassmorphism subtil
- **Animations 3D** : Arrière-plan Three.js avec particules flottantes et sphère interactive
- **Animations fluides** : GSAP pour les transitions et révélations au scroll
- **Responsive** : Optimisé mobile et desktop
- **Stack** : React, Three.js, GSAP, TailwindCSS

## 🚀 Installation

```bash
npm install
npm run dev
```

Le site sera accessible sur `http://localhost:5173`

### Accès depuis ton téléphone (même réseau Wi‑Fi)

1. Lance `npm run dev` sur ton Mac.
2. Dans le terminal, Vite affiche une ligne **Network** du type : `http://192.168.x.x:5173`
3. Sur ton téléphone, ouvre **cette adresse** (pas localhost) dans le navigateur, par ex. :  
   `http://192.168.1.42:5173/#hero`
4. Si ça ne marche pas : désactive temporairement le pare-feu du Mac pour le réseau privé, ou autorise le port 5173.
